# planner

## API

### Steps to setup a project:

- Install typescript globally

```
npm i typescript -g
```

- Install tslint globally

```
npm i tslint -g
```

- Install gulp cli globally

```
npm i gulp-cli -g
```

- Run command to install packages in API folder:

```
npm i
```

### To run a project

```
gulp server
```

or just:

```
gulp
```

Server will start on localhost:3000
